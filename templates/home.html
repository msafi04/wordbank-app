My Word Bank

This is a project to learn Django, this webapps has a english word journal, a reading book list, a english dictionary using PyDictionary package and a fun riddles page which asks a random riddle from a json file.

There is a Search functionality implemented for searching keywords from the words and the book list stored in the database.

The user must login to enter new words/book list as well as to edit the entries, also only logged in users can access riddles and dictionary pages however any user can view the words journal and the book list.
